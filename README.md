# Reusable Offline-First App Frameset

This repository defines a reusable architecture for Android, iOS, and optional web apps that can run independently with local storage and optionally sync to central team storage.

Start with:

- `docs/architecture.md` for the full module blueprint
- `docs/api-contracts.md` for backend and client-facing API contracts
- `docs/security-devops.md` for security, operations, CI/CD, and deployment practices
